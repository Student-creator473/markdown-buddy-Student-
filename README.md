# Markdown Buddy – R Project Documentation

## Project Overview

This project demonstrates how to create professional documentation for an R data analysis project using Markdown. The project includes a README file, an R Markdown (.Rmd) documentation file, and AI-assisted documentation following GitHub best practices.

## Installation

1. Install R from https://cran.r-project.org/
2. Install RStudio.
3. Install the required packages.

```r
install.packages("tidyverse")
install.packages("ggplot2")
```

## Example Usage

```r
source("analysis.R")
```
