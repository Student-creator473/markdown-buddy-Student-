# Reflection

## 1. How did AI help you complete this assignment?

AI helped me organize the README file, improve Markdown formatting, create the R Markdown documentation, and provide examples of proper GitHub documentation. It saved time and helped me produce a professional project structure.

## 2. What did you verify after using AI?

I reviewed the generated Markdown syntax, checked that headings, bullet lists, and code blocks displayed correctly in GitHub, and verified that the documentation matched the assignment requirements.

## 3. What are the strengths and weaknesses of using AI for documentation?

### Strengths

- Saves time.
- Improves writing quality.
- Produces professional formatting.
- Helps organize documentation.

### Weaknesses

- AI-generated content must always be reviewed.
- Small formatting mistakes may occur.
- Users remain responsible for verifying accuracy.
